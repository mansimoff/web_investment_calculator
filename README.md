## 🧮 Что умеет калькулятор

Один инструмент — пять режимов расчёта сложного процента. Достаточно ввести
известные параметры, а неизвестный калькулятор найдёт сам.

| Режим | Вопрос, на который отвечает |
|---|---|
| 💰 **Конечная сумма** | Сколько будет на счёте через N лет? |
| 📊 **Необходимый процент** | Какая ставка нужна, чтобы достичь цели? |
| ⏳ **Срок достижения цели** | Через сколько лет накопится нужная сумма? |
| 🔄 **Размер пополнений** | Сколько откладывать каждый месяц/квартал/год? |
| 💵 **Стартовый капитал** | Сколько нужно внести сейчас, чтобы достичь цели? |

Дополнительно:
- 📅 Гибкая периодичность пополнений — раз в месяц, квартал или год
- 📈 Интерактивный график роста капитала (баланс / вложено / проценты)
- 📄 Экспорт подробного отчёта в **PDF** — с графиком и таблицей по годам
- 📊 Экспорт в **Excel** — сводка + полная динамика по годам на отдельном листе
- ✅ Валидация формы на лету с понятными сообщениями об ошибках

## 🧠 Математика

Базовая единица расчёта — **месяц**. Введённая годовая ставка переводится
в эффективную месячную через:

```
r_месяц = (1 + r_год)^(1/12) − 1
```

Это даёт точное помесячное капиталирование, а не грубое деление ставки на 12.
Для режимов «процент» и «срок» используется бинарный поиск с точностью
до 0.01%, для «пополнения» и «стартовый капитал» — аналитические формулы
с перепроверкой через прямую симуляцию.

## 🚀 Запуск

Проект не требует сборки — достаточно открыть `src/index.html` в браузере
или поднять любой статический сервер:

```bash
# вариант 1 — VS Code Live Server
# просто открой src/index.html правой кнопкой → "Open with Live Server"

# вариант 2 — Python
cd src && python3 -m http.server 8080

# вариант 3 — Node.js
npx serve src
```

Затем перейти на `http://localhost:8080`.
